-Tecnologías y Conceptos Aplicados

1. HTML5 Semántico
Se ha construido una estructura robusta utilizando etiquetas semánticas para mejorar la accesibilidad y el SEO del sitio.
Estructura Principal: Uso de etiquetas <header>, <main>, <section> y <footer>.
Secciones Específicas: Implementación de <article> para la biografía, <aside> para datos complementarios y tarjetas de contacto, y <figure> con <figcaption> para la galería de proyectos.
Tablas y Formularios: Uso de <table> para el desglose de habilidades y un formulario completo con <input>, <select> y <textarea>.

2. CSS3 Moderno

El diseño se basa en un archivo CSS externo que aplica conceptos avanzados de estilizado.
Variables CSS (Custom Properties): Definición de un esquema de colores (azul principal, naranja, fondos grises) y tipografías en el bloque :root para facilitar la mantenibilidad.
Box Model: Aplicación del reset universal box-sizing: border-box para un control preciso de dimensiones, márgenes y rellenos.
Layouts Avanzados:
Flexbox: Utilizado para la alineación del encabezado, el centrado del Hero y la organización de las tarjetas de habilidades y contacto.
CSS Grid: Implementado en la sección de proyectos mediante grid-template-columns con auto-fit y minmax para lograr una cuadrícula responsiva sin necesidad de múltiples media queries.Efectos Visuales: Uso de linear-gradient para fondos y barras de progreso, además de transition y box-shadow para mejorar la experiencia de usuario (UX) en los estados :hover.📂 

-Estructura de Entregas (Commits)

El proyecto ha sido desarrollado siguiendo una progresión lógica de construcción por capas:

Commit 1: Configuración inicial, Reset CSS y estructura de Navegación con Flexbox.
Commit 2: Sección Sobre Mí utilizando estructura de dos columnas (article + aside).
Commit 3: Sección Habilidades con tabla de niveles y listas de competencias.
Commit 4: Galería de Proyectos implementando CSS Grid y figuras semánticas.
Commit 5: Formulario de Contacto y Footer profesional.
